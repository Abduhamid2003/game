## О проекте

Данный проект представляет собой **RESTful API-приложение** для игровой платформы, реализованное на современном фреймворке **FastAPI**.

Система предназначена для управления играми, уровнями, прогрессом игроков и системой достижений.  
Приложение может использоваться как backend для веб- или мобильной игровой платформы, где игроки проходят уровни, набирают опыт, повышают уровень и получают достижения.

---

## Основной функционал

Приложение предоставляет следующие возможности:

- Регистрация и аутентификация игроков (JWT)
- Профиль игрока: уровень, опыт, дата регистрации
- Управление играми (создание, просмотр)
- Управление уровнями игр (номер, название, награда в опыте)
- Управление достижениями (глобальные и привязанные к игре)
- Прохождение уровней → автоматическое начисление опыта
- Автоматическое повышение уровня игрока (по формуле)
- Автоматическое открытие достижений при достижении условий (уровень / опыт)
- Просмотр своих достижений
- Интерактивная документация API (Swagger UI / ReDoc)

## Возможности API

### Аутентификация
- `POST /auth/register` — регистрация игрока
- `POST /auth/login` — вход и получение JWT-токена

### Игрок
- `GET /igrok/profil` — профиль текущего игрока (уровень, опыт)
- `GET /igrok/dostizheniya` — список полученных достижений
- `POST /igrok/proiti-uroven` — пройти уровень → получить опыт и возможные достижения

### Игры
- `POST /igry/` — создать игру
- `GET /igry/` — список всех игр

### Уровни
- `POST /urovni/` — создать уровень для игры
- `GET /urovni/igra/{igra_id}` — уровни конкретной игры

### Достижения
- `POST /dostizheniya/` — создать достижение
- `GET /dostizheniya/` — все достижения
- `GET /dostizheniya/igra/{igra_id}` — достижения игры
- `GET /dostizheniya/globalnye` — глобальные достижения

## Как запустить проект локально

### Требования
- Python 3.10 или выше
- Git

### Пошаговая инструкция

```bash
# 1. Клонировать репозиторий
git clone https://github.com/ВАШ_ЛОГИН/gaming-platform-api.git
cd gaming-platform-api

# 2. Создать виртуальное окружение
python -m venv venv

# 3. Активировать окружение
# Windows:
venv\Scripts\activate
# Linux / macOS:
source venv/bin/activate

# 4. Установить зависимости
pip install -r requirements.txt

# 5. Запустить сервер
uvicorn main:app --reload --port 8001
